# Find Frequency of Number in Array


## Question
Given an array and queries, report occurrence count of queried number.

Source:
- https://www.geeksforgeeks.org/find-frequency-number-array/

## Testcases
- Input:
  `arr=[1,2,2,3], x=2`
  Output:
  `2`
  Explanation:
  Number `2` appears twice.

## Edge Cases
- Number not present.
- All elements identical.

## Constraint
- As per source problem statement.

## Hint
- Use hashmap or fixed frequency array.

## Solution
Count frequencies once and answer query in O(1) average.
